# Apollo
Материаллы по запуску внешнего кода на Apollo 18 in 1 B0302 для [статьи на Habr](https://habr.com/ru/articles/955488/)

**[Tapollo](https://github.com/azya52/Apollo/tree/main/tapollo)**<br />
Проект для RP Pico 2, позволяющий использовать ее в качестве внешнего последовательного ПЗУ для Apollo. Pico определяется как com-порт на который можно отправлять данные для передачи (без каких-либо заголовков). Передача начинается автоматически после перехода Apollo в "отладочный режим". **[Bad Apollo](https://github.com/azya52/Apollo/tree/main/badapollo)**<br />
Проект для RP Pico 2, включающий код и данные для демо [Bad Apollo](https://youtu.be/QnQHM4NFRGk). Воспроизведение начинается после перехода Apollo в "отладочный режим".
